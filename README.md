# Citation Manager — Citation Management SaaS

Manage business citations across 1000+ online directories. A modern, API-first alternative to Uberall, Bright Local, and Yext.

## Status
MVP in progress (48-hour sprint). See [GitHub Issues](https://github.com/michaelmonetized/citation-manager/issues).

## Tech Stack
- **Frontend:** Next.js 16, React 19, Clerk, Tailwind v4
- **Backend:** Convex
- **Infrastructure:** Vercel, Fly.io, Stripe

## Quick Start
```bash
bun install
bun run dev
```
